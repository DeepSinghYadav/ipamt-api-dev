# Payout System (Phase 4)

The Payout System manages agent payouts and financial settlements.

## Features (Planned)
- Payout cycle (weekly/monthly)
- Payout approval flow
- Payout ledger
- Bank details + KYC
- Payout history

## Business Rules
- Only verified commissions are payable
- Payout requires admin approval
- KYC mandatory for withdrawal

## Future Enhancements
- Auto-payout via payment gateway
- GST/TDS integration

# Sales Module (Phase 2)

The Sales Module handles booking, verification, and customer onboarding.

## Features (Planned)
- Sales entry API
- Booking request flow
- Customer KYC
- Document upload
- Booking verification

## Business Rules
- Unit must be available before booking
- Verification required before marking as sold
- KYC mandatory for final approval

## Future Enhancements
- Booking cancellation
- Refund workflow

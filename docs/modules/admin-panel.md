# Admin Panel (Phase 5)

The Admin Panel provides a UI for managing the entire system.

## Features (Planned)
- Dashboard (sales, commissions, team stats)
- Team tree UI
- Property management UI
- Sales verification UI
- Commission UI
- Settings UI
- Activity logs

## Business Rules
- Only admin can modify system settings
- Only admin can approve payouts
- Only admin can verify sales

## Future Enhancements
- Role-based access control (RBAC)
- Multi-admin support

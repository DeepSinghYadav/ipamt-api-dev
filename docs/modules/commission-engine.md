# Commission Engine (Phase 3)

The Commission Engine calculates multi-level commissions for agents.

## Features (Planned)
- Commission rules (admin controlled)
- Level limit (e.g., 3-level, 5-level)
- Multi-level commission calculation
- Commission logs
- Reversal/refund policy

## Business Rules
- Commission depends on sale amount
- Each level has a percentage
- Level limit configurable by admin

## Future Enhancements
- Builder-wise commission slabs
- Commission analytics dashboard
